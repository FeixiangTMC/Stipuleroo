# Stipuleroo

托叶工具，Minecraft 基岩版客户端多功能模组。纯客户端工作，可进入任意服务器使用。

## 功能

### 灵魂出窍 `/fc`

输入 `/fc` 或按快捷键（默认 `C` 键）切换灵魂出窍模式（旁观视角自由飞行）：

| 提示 | 含义 |
|---|---|
| `灵魂出窍模式已启用。` | 已开启，可自由飞行 |
| `灵魂出窍模式已禁用。` | 已关闭，回到身体 |

- 死亡、断线、切换世界时自动退出
- 客户端拦截发包，服务端无感知
- 快捷键可在 `config/config.json` 中修改

### 自动工具 `/at`

输入 `/at` 切换自动工具模式：

| 提示 | 含义 |
|---|---|
| `自动工具切换已启用。` | 已开启 |
| `自动工具切换已禁用。` | 已关闭 |

- 挖掘方块时自动从快捷栏中选择挖掘速度最快的工具
- 长按横扫到不同方块时自动切换对应工具
- 攻击实体时自动切换到伤害最高的武器
- 掉线、切换世界时自动关闭

### 自动搭路 `/ab`

输入 `/ab` 进入准备模式，再按快捷键（默认 `R` 键）开关自动搭路：

| 提示 | 含义 |
|---|---|
| `R键准备就绪，按R键开关自动搭路。` | 准备中，等待按键 |
| `自动搭路准备已取消。` | 退出准备模式 |

- 开启后自动在脚下和前方放置方块
- 快捷键可在 `config/config.json` 中修改

### 伪潜行（隐藏）

始终保持潜行状态，无需按住潜行键。

### 夜视（隐藏）

永久夜视效果，无需药水。

## 配置

模组首次加载后会在 `config/config.json` 生成配置文件：

```json
{
    "freecamKey": "C",
    "autoBridgeKey": "R"
}
```

支持的键名：`A`~`Z`、`0`~`9`、`F1`~`F12`、`Space`、`Tab`、`Enter`、`Shift`、`Ctrl`、`Alt` 及左右变体。

## 安装

1. 安装 [LeviLauncher](https://github.com/LiteLDev/LeviLauncher) 启动器
2. 在 LeviLauncher 中安装 LeviLamina 客户端
3. 将 `Stipuleroo-windows-x64.zip` 导入 LeviLauncher 启动器
4. 启动游戏，在聊天栏输入命令以启用功能
