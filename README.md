# Stipuleroo

托叶工具，Minecraft基岩版客户端多功能模组。纯客户端工作，可进入任意服务器使用。

## 功能

### 灵魂出窍 `/fc`

输入 `/fc` 切换灵魂出窍模式（旁观视角自由飞行）：

| 提示 | 含义 |
|---|---|
| `灵魂出窍模式已启用。` | 已开启，可自由飞行 |
| `灵魂出窍模式已禁用。` | 已关闭，回到身体 |

- 死亡、断线、切换世界时自动退出
- 客户端拦截发包，服务端无感知

### 自动工具 `/at`

输入 `/at` 切换自动工具模式：

| 提示 | 含义 |
|---|---|
| `自动工具切换已启用。` | 已开启 |
| `自动工具切换已禁用。` | 已关闭 |

- 挖掘方块时自动从快捷栏中选择挖掘速度最快的工具
- 长按横扫到不同方块时自动切换对应工具
- 掉线、切换世界时自动关闭

## 安装

1. 安装 [LeviLauncher](https://github.com/LiteLDev/LeviLauncher) 启动器
2. 在 LeviLauncher 中安装 LeviLamina 客户端
3. 将 `Stipuleroo.zip/` 文件导入 LeviLauncher 启动器
4. 启动游戏，在聊天栏输入命令以启用功能
